4 Machine Learning models using Linear Regression, Support Vector Machines (SVM), K-Nearest Neighbours (KNN) and K-Means Clustering are outlined with example use cases.

This is a mini-project completed during Lockdown with the aim to improve my Machine Learning knowledge and better understand when using a specific model is preferable over another.

Linear Regression
The aim of using this model was to implement a linear regression algorithm to predict students final grade based on a series of attributes (integer attributes), using the Student Performance dataset from UCI Machine Learning Repository.
This method attempts to model the relationship between two variables by fitting a linear equation to observed data. One variable is considered to be an explanatory variable, and the other is considered to be a dependent variable. It attempts to fit a best-fit line to the data points. Therefore, linear regression isn't really suited with randomly spread data points.

SVM
The aim of using this model was to use SVM to classify if a cancer cell is malignant or benign, using the Breast Cancer dataset from Sklearn
SVM's are typically used for classification tasks. They work very well for high dimensional data and allow for us to classify data that does not have a linear correspondence. A SVM has a large list of applicable uses. However, in machine learning it is typically used for classification. It is a powerful tool that is a good choice for classifying complicated data with a high degree of dimensions(features).

KNN
The aim of using this model was to use K-Nearest Neighbours algorithm to classify cars into 4 categories based upon certain features, using the Car Evaluation Data Set from the UCI Machine Learning Repository.
KNN's are also used for classification tasks. Rather than coming up with a numerical prediction such as a students grade or stock price it attempts to classify data into certain categories. The algorithm works best when we can identify clusters of points in the dataset, and so typically does not work as well with high-dimensional data.

K-Means Clustering
The aim of using this model was to implement a a K Means algorithm to classify hand written digits, using the Digits dataset from Sklearn.
K Means clustering is an unsupervised learning algorithm that attempts to divide our training data into k unique clusters to classify information. This means this algorithm does not require labels for given test data. It is responsible for learning the differences between our data points and determine what features determining what class.
